******************************
Running Snake Change Direction
******************************
======   ======   ============
Level    Points   Uses
======   ======   ============
Easy	 1	  LED Display
======   ======   ============

Make an LED snake 3 pixels long run around the outside of the LED display. The snake should change direction if button `A` is pressed.

You will need
=============

 * Micro:bit
 * USB cable
